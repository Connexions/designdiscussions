Archive Design

Coversion of CollXML to HTML

  • CollXML is converted using the collxml2html.xsl file
  • The resulting CollXML and metadata are stored in the Archive database


CNXML to HTML Conversion

  • Use code from cnx-transforms, rhaptos.cnxmlutils and oer.exports
  • Has a mass conversion script that works directly against the database and can be run more than once.
  • Uround trip (published from cnxml to html), but not guarantee html to cnxml. cnxml is raw cnxml that points to legacy urls for resources and such.
