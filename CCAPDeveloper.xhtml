Community College Access Project (CCAP) Developer Job Description


Connexions , a leading publisher of free, open-licensed textbooks and learning materials, is looking for a developer to help us provide production-quality textbooks on the web and in print (PDF) along with some web design help.  Contract is full time for 9 months with a possible extension.

Job Requirements
  • Modify existing HTML/CSS to match design of additional textbooks
    • This is not simple CSS/HTML as used in a static HTML web site.
    • See current example code at https://github.com/Connexions/oer.exports/tree/master/css
  • Work with Connexions team to create design for Backbone.js based web application

Skills Needed
  • Required
    • Strong expertise in CSS2  and CSS3
    • Experience with JavaScript
    • Experience developing designs that work well with Backbone.js
  • Nice to have
    • Experience with Docbook’s XSLT conversions
    • Experience using LessCSS

If interested, send your resume to jobs@cnx.org referencing CCAP Developer in the subject.
