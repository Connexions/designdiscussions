JavaScript/Web Developer Job Description


Connexions, a leading publisher of free, open-licensed textbooks and learning materials, is looking for a contract software developer to help with the rewrite of Connexions in JavaScript, HTML5 and Python.  Full time contract is for 6 months with a possible extension. We are open to having someone remote for this position. Connexions is an open source project.  Our code is on Github and we are doing Agile development.


Job Description

  • Develop site components using various JavaScript packages
    • Backbone.js
    • Bootstrap
    • Marionette
    • Underscore.js
    • MustacheJS
    • Require.js
    • jQueryUI
  • Develop templates as needed.
  • Work with Connexions team to design components for site

Requirements
  • Strong programming experience in MVC JavaScript.
  • Experience using RESTful APIs
  • Experience developing single page applications
  • Nice to have
    • Python programming experience
