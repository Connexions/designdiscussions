What is Authoring Services?

As part of the Connexions Rewrite, Authoring Services has three main components

  • An almost WYSIWYG HTML 5 Editor that allows authors to enter their content and quickly see the visual results.
  • A Workspace that displays all the content a user has created or that has been shared with them.  The content can organized into folders and viewed by folder.
  • An Unpublished Content Repository (Unpub Repo) where content is stored before it is published in the Connexions Archive (published content repository).

What problems exist now?

The current Authoring system in Connexions is powerful, but limiting.  If you are technically able to create your content using Latex or editing XML, you can create beautiful compelling textbooks.  For the non-technical, the options in the current system can intimidate and keep authors that want to use Connexions from adding to our repository.  Connexions has 3236 published authors, but has content that has never been published created by 5121 users. There appears to be a pattern of users trying out the current authoring tools and never completing their work.  The current system also does not provide any feedback to an author when export files (PDF, EPUB, etc.) are generated or when the generation fails.

How does it solve these problems?

Authoring Services solves the authoring issues by allowing authors to enter content using an editor similar to editors authors are familiar with, provides feedback on the publishing process and provides a Workspace that has simple organization and sharing.  Improved performance for all of this functionality will make authoring a much smoother, more pleasant experience.
