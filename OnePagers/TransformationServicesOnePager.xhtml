What is Transformation Services?

As part of the Connexions Rewrite, Transformation Services will be a suite of import and export services used by Connexions as part of editing and publishing.


Service Used By
PDF Generation Editor, Publishing
EPUB Generation Publishing
Mobi Generation Publishing
Complete ZIP Generation Publishing
HTML Export Publishing
CNXML Export Publishing
Word document Importer Editor
Latex Importer Editor
Zip Importer Editor
Sword Interface Editor

What problems exist now?

Currently, many of these services are available as part of Connexions and some books can take several hours to generate export files when a collection is published.  Because of the long running processes, they can sometimes impact the performance of the Connexions web site.  The services are tightly coupled with Connexions which makes it difficult for outside groups to access them.

How does it solve these problems?

Moving all of these services to a separate servers and components will isolate the performance issue without impacting the Connexions web site.  The services will be loosely coupled and run asynchronously from the Connexions web site. Because they will be exposed via a RESTful API, they can be used by other systems using the Connexions Author Services components or Rhaptos.  The services will have monitoring capabilities which will allow services to be individually monitored and turned on and off as needed.

Release

Transformation Services will be released in early 2013.
