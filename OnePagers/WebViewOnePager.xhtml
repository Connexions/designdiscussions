Web View One Pager

What is the Web View?

As part of the Connexions Rewrite, the Web View is the display of Connexions content.  It includes modules, collections, author profiles, lenses, search and help pages.


Connexions currently has approximately 1 million unique visitors a month.  Most of these visitors find us via a search result from Google, Bing or Yahoo.  Approximately 13,000 unique users visit the Connexions Search page and perform about 34,000 searches each month. Lenses are viewed about 8500 times per month.  Modules are viewed 950,000 times a month and collections 150,000 times by unique visitors.  There are no stats on the usage of our mobile site.

What problems exist now?

The mobile and browser version of the site are separate without any shared infrastructure which results in incomplete metrics on site usage. In the browser version of the site, there is too much information displayed at once.  All of the information is important, but it does not need to be available all of the time.  Portlets such as Lenses, Recently Viewed and Tags surround the content and distract the user from their main purpose which is reading the content.

How does it solve these problems?

The Rewrite Web View will have a simple design that allows the user to focus on their main tasks which are search and viewing content.  All of the data available now on the Web View will still be available.  It just will not be displayed all at once.  The site will also have one version of the site for all size devices via use of responsive design.  This will allow the gathering of metrics on user engagement on all devices.

Release

The Rewrite Web View will be released in Q2 of 2013.
