CNX Rewrite Logging Documentation

The logging system has 2 main components

  • Syslog
  • Graphite

Syslog

  • Syslog allows for a centralized logging system
  • All events will be logged in the same log
  • Logs will roll when they are 10 MB or daily, whichever comes first

Graphite

  • Parses log files and displays results in various ways

For more information see detailed documentation
