Connexions Rewrite Design Decisions

This document captures the design decisions made during the daily Connexions design meeting.

Math Editing

  • We will support entering math using
    • LaTeX
    • MathML
    • Maybe ASCII Math
  • If math is imported using the LaTeX importer, we will attempt to grab the math and store it in a microdata format in the HTML.  We will key off the MathMode in LaTeX to locate the math.
  • The Editor will display math as rendered by MathJax.  When a user clicks on an Equation, a textbox will open to allow them to edit the MathML or LaTeX.

Authoring Repository

Modules

  • Store as HTML on file system
  • Path to file on file system will be stored in database
  • Metadata stored in file as MicroData
  • User ids only are stored in the metadata.  Other info in the OAuth repository.

Collections

  • Store as HTML or JSon on file system
  • Path to file on file system will be stored in database
  • Metadata stored in file as MicroData
  • User ids only are stored in the metadata.  Other info in the OAuth repository.

My Stuff

  • List of content
  • tagging to filter content
  • Share content with others
  • Store in the database
