Archive Search Design

Simple Search


Simple Search will use Lucene syntax

Examples


Search for words chemistry and organic:

cnx.org/search?q=”chemistry organic”


Search for author Ed Woodward


cnx.org/search?q=”author:’Ed Woodward’”


Search for author Ed Woodward with Book type selected


cnx.org/search?q=”author:’Ed Woodward’ type:book”


Advanced Search


Advanced search will use Lucene syntax: https://lucene.apache.org/core/3_5_0/queryparsersyntax.html


Example


Search for:

     keywords chemistry and organic

     author “John”  with these filters:

  • Subject:Science and Technology

  • Language:any (default)

  • Books only

  • Publication Date: any (default)

  • Sort by: Publication Date


cnx.org/search?q=”keyword:chemistry keyword:organic  subject:’Science andTechnology’ sort:’Publication Date’ type:book author:John”


<first result set w/ 5000 results elided>

Then click on the additional limit associated w/ UUID1 (perhaps John Smith :-) )


cnx.org/search?q=”keyword:chemistry keyword:organic  subject:’Science and Technology’ sort:’Publication Date’ type:book author:John; authorID:UUID1”
